# Проект "MAKE_BANK"

## Описание:
Проект "MAKE_BANK" представляет собой приложение для управления банком, которое позволяет пользователям выполнять различные операции, такие как открытие и закрытие счетов, пополнение баланса, перевод средств и просмотр истории транзакций. Основная цель приложения - предоставить удобный и безопасный интерфейс для работы с финансовыми операциями.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/iPavels/make_bank.git
```
2. Использование функций:
```
get_mask_card_number, get_mask_account, filter_by_state, sort_by_date, mask_account_card, get_date
```
## Использование:
```
1. get_mask_card_number - Функция принимает на вход номер карты и возвращает ее маску.
2. get_mask_account - Функция принимает на вход номер счета и возвращает его маску.
3. filter_by_state - Принимает список словарей и значение для ключа state.
4. sort_by_date - Принимает список словарей и еще один параметр и сортирует. 
5. mask_account_card - Функция принимает строку с типом и номером карты или счета, возвращает замаскированный номер.
6. get_date - Функция возвращает дату.
```

## Тестирование:
Проект включает в себя автоматизированные тесты, расположенные в папке /tests/, которые проверяют корректность работы основных функций.


## Запуск тестов
```
pytest
```
Если требуется подробный вывод:
```
pytest -v
```
# Покрытие кода тестами

Чтобы проверить покрытие кода тестами, используйте:
```
pytest --cov
```
